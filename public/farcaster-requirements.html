Requisitos para que el SDK de Farcaster funcione correctamente

¡Importante! Esta página explica por qué la aplicación necesita ser cargada dentro de la app de Warpcast para que funcione el SDK de Farcaster.

¿Por qué no funciona en el navegador normal?

El SDK de Farcaster está diseñado para funcionar específicamente dentro del entorno de Warpcast como una mini app. Cuando accedes desde un navegador normal, el SDK no está disponible porque:

  • El SDK es inyectado por la aplicación Warpcast en el contexto de su navegador interno
  • El SDK requiere autenticación y contexto de Warpcast para funcionar
  • Las mini apps de Farcaster solo funcionan cuando son cargadas dentro del navegador de Warpcast

Cómo probar la aplicación correctamente

  1. Accede a la aplicación Warpcast en tu dispositivo móvil
  2. Busca la mini app "LottoMoji" o utiliza el enlace directo: https://warpcast.com/miniapps/MD6NcmUnNhly/lottomoji
  3. La aplicación se abrirá dentro de Warpcast, con el SDK disponible
  4. Ahora podrás generar tickets y firmar transacciones con tu cuenta de Warpcast

Causas comunes de error

  • Acceso directo desde navegador: Cuando accedes directamente a https://lottomojifun.vercel.app desde un navegador normal, el SDK no está disponible.
  • Navegador dentro de Warpcast: Si abres un enlace externo dentro de Warpcast que no es una mini app registrada, tampoco funcionará.
  • Permisos de la mini app: La mini app debe estar correctamente registrada en Warpcast y tener los permisos adecuados.

Solución para desarrollo y pruebas

Si necesitas probar la aplicación durante el desarrollo:

  • Puedes utilizar los "modos de emergencia" implementados que crean usuarios genéricos
  • Para pruebas reales, siempre debes usar la app de Warpcast en un dispositivo móvil
  • No hay forma de simular completamente el SDK de Farcaster fuera de Warpcast

Nota técnica: El SDK de Farcaster se carga como un objeto global en el navegador de Warpcast. La mayoría de los errores de "SDK no disponible" ocurren porque se está accediendo fuera del contexto de Warpcast.
